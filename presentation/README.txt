PRESENTATION FOLDER

This folder contains your final project presentation materials.

FILES INCLUDED
--------------
1. slides.md - Presentation content in Markdown format

TO CREATE YOUR SLIDES
---------------------
Option A: Convert to PowerPoint/Google Slides
   - Copy each section (separated by ---) into individual slides
   - Use a dark theme to match the RPG aesthetic

Option B: Use Marp (Recommended for developers)
   - Install Marp CLI: npm install -g @marp-team/marp-cli
   - Convert to PDF: marp slides.md --pdf
   - Convert to PPTX: marp slides.md --pptx

Option C: Use reveal.js
   - Copy content to reveal.js HTML template
   - Host locally or export

PRESENTATION STRUCTURE (8-10 minutes)
-------------------------------------
1. Introduction (1 min)
   - Project overview
   - Personal motivation

2. Technical Demo (3-4 min)
   - Live demonstration of game
   - Show all 4 quests
   - Architecture overview

3. Technical Deep Dive (1-2 min)
   - Challenges faced
   - Innovative solutions

4. Conclusion (1 min)
   - Achievements
   - Future improvements
